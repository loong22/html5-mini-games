## ✨ 项目简介

**HTML5 Mini Games** 是一个完全使用 **原生 HTML、CSS、JavaScript** 构建的网页小游戏合集。
所有游戏均存放在 `games/` 文件夹下，每一个都是 **独立、简单、可直接运行** 的小游戏。

特点如下：

* 🌱 无任何框架或第三方依赖（纯原生）
* 🎮 每个游戏都是独立的小项目
* ⚡ 双击即可运行，无需构建、无需安装
* 📚 适合作为前端初学者练习项目
* 🧪 可作为 Web 动画与交互实验场
* 🔧 易于扩展，可自由添加新游戏

---

## 📁 项目结构

```
/games
  /game1
    index.html
    style.css
    script.js
  /game2
    index.html
    style.css
    script.js
assets/
README.md
```

* ✔ 所有游戏都位于 `games/` 目录
* ✔ 结构规则统一，便于扩展
* ✔ 每个游戏由 3 个文件构成，可直接在浏览器运行

## 🚀 如何运行

运行任意游戏非常简单：

1. 克隆或下载仓库
2. 进入 `games/` 目录
3. 打开任意游戏文件夹
4. 双击 `index.html`
5. 浏览器自动打开即可开始游戏 🎉

📌 *无需服务器、无需 npm、无需构建工具。*

若你喜欢，也可以使用：

* VS Code + Live Server
* GitHub Pages
* Cloudflare Worker/Pages

---

## ✏️ 如何添加新游戏

在 `games/` 目录下创建一个新文件夹，例如：

```
games/my-new-game/
  index.html
  style.css
  script.js
```

然后在 README 中加入游戏名称即可。
